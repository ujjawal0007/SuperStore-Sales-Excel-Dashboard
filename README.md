## 📊 Project Overview

As part of my Data Analytics journey, I built a complete **interactive Sales Dashboard in Microsoft Excel** for a Super Store retail business.

The dashboard helps business stakeholders:
- 💰 Track total sales and revenue performance
- 🚚 Understand shipping mode preferences
- 🌍 Compare region-wise transaction volumes
- 🏙️ Identify top-performing cities
- 📦 Analyze category-wise and consumer-type trends

---

## ✅ Dashboard Features

### 1. 🔢 KPI Cards
- **Total Sales** — 5,820,182
- **Sales Average** — 582.37
- **Total Transactions** — 9,994
- **Total Quantity** — 25,123

### 2. 🎛️ Interactive Filter Panel
- **Region Filter:** Central | East | South | West
- **Year Filter:** 2014 | 2015 | 2016 | 2017

### 3. 🚢 Ship Mode Sales %
- Pie chart showing:
  - Standard Class — 58%
  - Second Class — 20%
  - First Class — 16%
  - Same Day — 6%

### 4. 🌍 Region Wise Transactions
- Donut chart comparing Central, East, South & West regions

### 5. 📅 Monthly Quantity Ordered
- Line chart tracking quantity ordered Jan–Dec
- Helps identify **seasonal demand peaks**

### 6. 🏙️ Top 5 Cities Sales Wise
- Bar chart — top performing cities:
  - New York | Los Angeles | Philadelphia | Seattle | San Francisco

### 7. 📦 Category Wise Transactions
- Bar chart:
  - Office Supplies — 6,026
  - Technology — 1,847
  - Furniture — 2,121

### 8. 👥 Consumer Type Quantity Ordered
- Bar chart:
  - Consumer — 13,030
  - Corporate — 7,598
  - Home Office — 4,495

## ▶️ How to Use

   ```bash

   ```

2. **Open the file**
   → `Dashboard.xlsx` in Microsoft Excel
